# Custom NCBI genome assembly statistics fetcher

## Dependencies
Entrez

## Input
Taxonomic ID

## Output
Assembly statistics in .csv format, including fields:
Organism name
Assembly-ID, 
taxonomic ID, 
Genbank release date, 
Coverage, 
Assembly status, 
ContigN50, 
ScaffoldN50, 
BioSampleAccn,
Isolate
